# 🍔 Mini Food Ordering Hub

A highly interactive, component-driven food ordering application built with React, Vite, and Tailwind CSS.

## 🚀 Features
* **Dynamic Menu Grid:** Renders culinary selections instantly from state data.
* **Smart Search & Filters:** Filter menu items seamlessly by categories (Pizza, Burgers, Sushi).
* **Sliding Cart Sidebar:** Manages persistent shopping item totals and quantity adjustments dynamically.
* **Local Storage Sync:** Cart data persists across browser reloads using dynamic state side-effects.
* **Modern Premium Theme:**  complete with smooth entry transitions.

## 🛠️ Tech Stack
* React.js (Hooks & Reusable Components)
* Vite (Build System)
* Tailwind CSS (Premium Layouts)
* Lucide React (Icon Packs)
